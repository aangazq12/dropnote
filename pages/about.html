About

What is DropNote?

DropNote adalah sistem catatan lokal untuk airdrop hunter dan Web3 task.

Semua data disimpan di perangkat kamu. Tidak perlu akun. Tidak ada sync paksa.

Why do I see example notes?

Saat pertama kali digunakan, DropNote menampilkan example notes sebagai panduan.

Contoh ini hanya muncul sekali dan aman untuk dihapus.

How notes work
  • Judul digunakan sebagai identitas catatan
  • Konten disimpan apa adanya
  • Tidak ada auto-format atau auto-correct

DropNote tidak mencoba “memperbaiki” catatan kamu.

About tags

Tag digunakan untuk mengelompokkan catatan seperti Airdrop, Testnet, atau Daily Task.

Home hanya menampilkan tag yang relevan sebagai ringkasan aktivitas.

What is LFG note?

LFG adalah catatan cepat untuk FCFS, whitelist, atau info sensitif.

Biasanya berisi:

  • Wallet (EVM / SOL)
  • Social ID (Twitter, dsb)

Tag FCFS hanya muncul di LFG dan tidak ditampilkan di Home.

Wallet notifications (Telegram)

DropNote dapat menampilkan notifikasi aktivitas wallet melalui Telegram.

Cara kerja:

  • Buat channel Telegram (private / public)
  • Tambahkan DropNote bot ke channel
  • Tambahkan bot pemberi sinyal (misalnya Etherdrop)

Catatan penting:

  • DropNote tidak memantau wallet secara langsung
  • Semua event berasal dari pesan Telegram
  • Tidak ada signing atau eksekusi transaksi
  • Hanya notifikasi (read-only)

Wallet notification adalah sinyal, bukan aksi.

Settings behavior

Settings memengaruhi tampilan dan perilaku aplikasi.

Perubahan settings tidak mengubah catatan lama.

Your data
  • Semua data disimpan lokal
  • Tidak perlu akun
  • Tidak ada server atau cloud
  • Backup manual via export
